# BlvckWinPipe
### Modern C++ IOCP-Based Windows Named Pipe Framework

WinPipe is a high-performance framework for building
scalable Windows Named Pipe servers and clients
using modern C++ and I/O Completion Ports (IOCP).

It provides a clean session-based architecture,
automatic lifetime management,
and efficient async read/write pipelines
designed for real production workloads.

## Tech Stack

- **Language:** C++20
- **Build system:** CMake 3.20+
- **Platform:** Windows 10+
